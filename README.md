# Terraform Azure Infrastructure

## Project Overview

This repository demonstrates how to provision Microsoft Azure infrastructure using **Terraform**, following Infrastructure as Code (IaC) best practices.

The project is designed to showcase Terraform fundamentals, Azure resource deployment, and version control using GitHub.

---

## Technologies Used

- Microsoft Azure
- Terraform
- Azure Resource Manager (AzureRM Provider)
- Git
- GitHub

---

## Azure Resources

This project will provision the following Azure resources:

- Resource Group
- Virtual Network (VNet)
- Subnet
- Network Security Group (NSG)
- Storage Account
- Virtual Machine
